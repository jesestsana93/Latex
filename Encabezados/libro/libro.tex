% headings - El encabezado contiene el número y nombre de sección en la parte izquierda y el número de

%            el encabezado está vacío y el pie contiene el número de página centrado.
% myheadings - El encabezado contiene el número de página a la derecha, el resto del encabezado puede ser 
%              definida por el usuario; para esto se pueden usar los comandos:
%              \markright - para documentos de clase book, report y article.
%              \markboth  - únicamente para documentos de clase book.
% Por ejemplo, las siguientes instrucciones colocan "Carlos Pérez" a la izquierda del encabezado,
